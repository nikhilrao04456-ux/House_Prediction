
# 🏠 House Price Prediction using Machine Learning

Predict house prices using machine learning regression models with end-to-end data preprocessing, feature engineering, model training, and evaluation.


## 📖 Overview

This project demonstrates a complete machine learning workflow for predicting house prices based on housing features. It covers everything from data cleaning to model evaluation using multiple regression algorithms.

## 🚀 Features

- 📊 Exploratory Data Analysis (EDA)
- 🧹 Data Cleaning & Missing Value Handling
- 🔧 Feature Engineering
- 📏 Data Scaling & Encoding
- 🤖 Multiple Regression Models
- 🔍 Hyperparameter Tuning
- 📈 Model Performance Comparison
- ✅ Cross Validation



## 🛠️ Tech Stack
